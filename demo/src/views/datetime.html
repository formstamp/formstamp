## Directive `fs-datetime`

  This directive creates a time-input widget. This directive expects
  that scope property specified in ngModel will be a Date,
  and will modify only hours/minutes/seconds components of this Date,
  year/month/day will remain unchanged. Supported attributes:





  
  

  

    
  












  

  

  
 

  

    
  






Time is: {{ svalue | json }}








  
  

  

    
  






Entered datetime: {{ selectedDateTime | date:'medium'}}
