Directive fsFormFor

This directive allows to construct complex horizontal forms with easy DSL-like markup. Used primarily for asynchronous submits. For synchronous use see fsSyncFormFor.

Each form row is described with fsInput directive. It has following attributes:

  • as - specifies type of input
  • label - specifies label text
  • name - specifies model's attribute to which input will be bound
  • Other attributes (like items) are directly passed into input directive.

Directive fsSyncFormFor

This directive is similar with fsFormFor. It's intended to use where synchronous form submit is needed. It is serialize entire form object to the json to _json hidden input.

  • sync-attr - Used for customizing name of hidden field. Default: _json.
