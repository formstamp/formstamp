Checkboxes

Directive fsCheckbox

This directive creates a set of checkboxes. List of available choices is passed via items attribute and can be an Array of Objects, Strings, Numbers, etc. User can select multiple items. fsCheckbox supports custom templating for item's label. Supported attributes:

  • items - property of scope containing list of available values (of strings for autocomplete, if freetext is enabled)
  • ng-model - see AngularJS documentation for ngModel directive
  • ng-disabled - disable/enable input with scope property
  • class - additional CSS classes
